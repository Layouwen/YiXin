## 介绍

易新小账本是一款简约风的记账应用。 从设计至完成，借鉴参考了多款 AppStore 中的类似应用。最终使其最简化。 本项目期间遇到很多问题，如 webpack配置 ，TypeScript 等问题。最终都记录在博客中。

## 二次开发

安装依赖

```
yarn install
```

本地预览

```
yarn serve
```

打包

```
sh deploy.sh
```
